What the Toolkit Shows

Average-SNR screening collapses probabilistic feasibility into hard decisions. It over-accepts canyon and moving-trail cases, then hard-rejects tent/shelter even though the LOS/NLOS mixture still exposes partial availability. A single-state lognormal model remains too optimistic for persistent NLOS states.

How to Use It

Run the lightweight command-line screen:

pip install -r requirements-lite.txt
python quick_run.py
python quick_run.py --scenario canyon --added-loss-db 2

Run the interactive dashboard locally:

pip install -r requirements-dashboard.txt
streamlit run app.py

Run the Python-only command-line workflow:

python run_all.py --skip-reference-cosim

Run the full reference workflow when MATLAB, Simulink, and Communications Toolbox are available:

python run_all.py
MIT-licensed research toolkit. The models use public-style proxy parameters and sensitivity sweeps, not proprietary handset calibration data.
